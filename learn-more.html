How to Make a Basic Website

Creating a basic website involves using three main technologies: HTML, CSS, and optionally JavaScript.

1. Structure with HTML

HTML (HyperText Markup Language) is used to create the structure of your website. A simple HTML page looks like this:

        <!DOCTYPE html>
        <html>
        <head>
            <title>My First Website</title>
        </head>
        <body>
            <h1>Hello, World!</h1>
        </body>
        </html>

2. Styling with CSS

CSS (Cascading Style Sheets) is used to style your website. Example:

        body {
            background-color: lightblue;
            font-family: Arial, sans-serif;
        }
        h1 {
            color: darkblue;
        }

3. Making it Interactive with JavaScript

JavaScript adds interactivity to your website. Example:

        <script>
            alert('Welcome to my website!');
        </script>

That's it! You've created a basic website.
